# RFC 101: note editor component

## 目的
学習ノートを編集するための UI コンポーネントを追加する

## 非目的
- API接続
- 自動保存
- リッチテキスト
- Markdown機能

## 受け入れ条件
- NoteEditor コンポーネントを追加する
- textarea を持つ
- label を表示できる
- disabled 状態を持つ
- story を追加する
- test を追加する
- frontend judge を通す

## 禁止事項
- page.tsx に直接 UI 実装を書かない
- story を追加せずにコンポーネントを追加しない
